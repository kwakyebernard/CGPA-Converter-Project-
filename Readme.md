# CGPA Converter

This is a simple web application that converts CGPA from 4.0 scale to 100-point scale.

## Features
- Converts CGPA (4.0 scale) to 100-point scale
- Real-time calculation
- Built with HTML, CSS, JavaScript

## How to Run
1. Open `index.html` in a browser
2. Enter your CGPA
3. Get the 100-point equivalent
